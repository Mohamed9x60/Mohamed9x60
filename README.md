# Tor VPN IP Changer

This script allows you to change your IP address using Tor, a free and open-source software that provides anonymizing communication.

## Prerequisites

Before running the script, make sure you have Python installed on your system. You also need to install the following dependencies:

```
pip install stem geoip2
```

## Usage

1. Clone the repository:

```
git clone https://github.com/your-username/tor-vpn-ip-changer.git
```

2. Navigate to the project directory:

```
cd TorVpn
```

3. Install the required dependencies:

```
pip install -r requirements.txt
```

4. Run the script:

```
python3 TorVpn.py
```

5. Follow the on-screen instructions to configure the VPN session duration and the interval between IP changes.

6. The script will automatically start the Tor service and change your IP address at the specified intervals.
